# Pipeline-Perplexity-App

Educational pipeline design tool for young engineers.

## How to use

1. Install Python 3.
2. Clone this repo.
3. Run:

   ```bash
   python calculations.py
   python my_first_case.py
## Non-hydraulic awareness notes

See `PIPELINE_NOTES_NON_HYDRAULIC.txt` for brief, non-calculation notes on:
- Route marking and sign boards
- External corrosion and coatings
- Basics of cathodic protection (CP)
- Pig launcher and receiver purpose
- Typical electrical & instrumentation items along a pipeline

These notes are for awareness and education only. They do not replace specialist design, codes or company standards.
